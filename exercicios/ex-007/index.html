Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM(semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

E aqui em baixo temos outro texto marcado.

Texto grande e pequeno

Podemos usar a tag BIG para deixar um texto grande. Essa tag foi depreciada , pois está ligada a forma e não há semântica.

Para deixar o texto pequeno usamos a tag SMALL.

Texto deletado

Podemos marcar um texto como deletado quando quisermos que ele seja lido mais não considerado.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois. Subustitui a tag u(sublinhado) não semântica.
